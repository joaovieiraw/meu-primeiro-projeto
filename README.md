Este projeto consiste em um sistema simples de gerenciamento de filas para atendimento em hospital, desenvolvido com HTML, CSS e JavaScript. O objetivo é organizar o atendimento de pacientes cadastrados em duas filas: fila normal e fila preferencial.
O sistema permite cadastrar pacientes informando nome, telefone e CPF. Após o cadastro, o paciente é adicionado à fila correspondente (normal ou preferencial). As filas são exibidas na tela e atualizadas automaticamente sempre que um paciente é adicionado ou chamado para atendimento.
O algoritmo de chamada segue uma regra de prioridade, onde até dois pacientes da fila normal podem ser chamados antes de um paciente da fila preferencial, garantindo um equilíbrio entre os atendimentos.
Além disso, o sistema mostra na tela qual paciente está sendo chamado, juntamente com seus dados, e mantém as listas de espera organizadas em tempo real.
Esse projeto demonstra conceitos importantes de programação como:

Manipulação do DOM
Uso de arrays
Estruturas condicionais
Funções em JavaScript
